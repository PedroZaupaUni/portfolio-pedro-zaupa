# Portfólio Pessoal — Pedro Zaupa

Este projeto é um portfólio pessoal desenvolvido como trabalho final da disciplina de Desenvolvimento Web.

O site apresenta uma página inicial com uma breve apresentação, uma página com projetos desenvolvidos e uma página de contato.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Estrutura do projeto

- `index.html`: página inicial
- `projetos.html`: página de projetos
- `contato.html`: página de contato
- `css/style.css`: arquivo de estilos
- `js/script.js`: arquivo com as funções JavaScript
- `img/`: pasta de imagens

## Observação

Alguns projetos profissionais são apresentados de forma resumida por envolverem informações privadas e repositórios não públicos.
